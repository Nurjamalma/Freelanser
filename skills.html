HTML (HyperText Markup Language)

Foundation for all web development starts with HTML.It uses standardised markups or tags that provide the essence for texts, images and many more.
The language is easy to learn; HTML uses unpaired or paired tags to render objects using DOM (Document Object Model) Interface. For example, if you were to create a page with a heading text, you could enter the following texts.

< !DOCTYPE html>

< html>

< head>

< title>My Story< /title>

< /head>

< body>

< h1>My Heading< /h1>

< p>This is my text.< /p>

< /body>

< /html>

Front-End Developers are often helpful to these indent lines, always keeping track of tags to close them correctly. However, it is essential to learn HTML for debugging

2) CSS (Cascading Style Sheets)

CSS allows front-end developers to give websites a unified appearance, a
premium feature for branded sites..

There is a style sheet inside the website file structure which reduces the code
you enter to help sites load faster and also to minimise the errors.

The style sheet structure can help to create a responsive design for users
whether they are visiting on a desktop or mobile phone

Here a front-end developer can control the visual aspects of a web page, such
as font colours, button design, and text alignment. This language is
straightforward, with many options for customization.

If you want to develop consistent-looking headers, for example, if you
navy blue H1 headers, you can enter the following code.

width{200px}

height{200px}
