  1. HTML (HyperText Markup Language)

Foundation for all web development starts with HTML.It uses standardised markups or tags that provide the essence for texts, images and many more.
The language is easy to learn; HTML uses unpaired or paired tags to render objects using DOM (Document Object Model) Interface. For example, if you were to create a page with a heading text, you could enter the following texts.

< !DOCTYPE html>

< html>

< head>

< title>My Story< /title>

< /head>

< body>

< h1>My Heading< /h1>

< p>This is my text.< /p>

< /body>

< /html>

Front-End Developers are often helpful to these indent lines, always keeping track of tags to close them correctly. However, it is essential to learn HTML for debugging
